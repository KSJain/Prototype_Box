# KSJ

Engineering should feel like magic.

KSJ is a collection of modular embedded libraries and reference hardware for building reliable, understandable, and reusable automation systems.

The ecosystem is designed around small, focused libraries that work independently and compose into larger products such as Prototype Box and RAMU Garden.
